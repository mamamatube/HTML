コード 名称 意味
<h(数字)> heading 見出しを表す。数が小さいほどサイズが小さくなる
<p> paragraph テキストの段落を書く際に用いる
<img> image 画像を配置する際に用いる
src="" ソース(source) 画像などの出所を表示する属性。""内にはファイル名を指定する
alt="" オルト(alternative) 画像が表示できなかった場合の代替テキストを指定する属性。
width="" ウィドゥス 要素の幅を指定する属性。
<body>に挿入する場合は自動的にpx表示となり""内には数字のみ指定する。
height="" ハイト 要素の高さを指定する属性。
<body>に挿入する場合は自動的にpx表示となり""内には数字のみ指定する。
<ul> Unordered List 箇条書きリストを表す。
<li> List Item リストの中身（項目）を表す。
<ol> Ordered List 順序付きリストを表す。項目には<ol>と同じく<li>を用いる。
<dl> Description List 説明付きリストを表す。項目には<dt>を用いる。
<dt> Description Term <dl>の項目を挿入する際に用いる。説明には<dd>を用いる。
<dd> Description Details <dt>の説明を表す。
<table> table テーブルリストを作成する際に用いる領域。
<thead> Table head テーブルリストの見出し領域を意味する。
<tbody> Table body テーブルリストの本体領域を意味する。
<tr> Table row row=行の意。
テーブルの表それぞれを行ごとに区切る役目を持つ。
<th> Table header <thead>内のセル作成に用いる。
<td> Table data <tbody>内のセル作成に用いる。
<a> anchor 囲った文字をリンクにする。
href="" エイチレフ属性 hypertext reference
<a>のリンク先を指定する。
target="" ターゲット リンク先の文書を開くフレームやウインドウを指定するための属性。
"_blank"を指定することで、クリックしたリンクが別タブで表示される。
<section> セクション それぞれの見出しがまとめている内容の範囲を区切るためのタグ。実行画面上に影響はない。
<nav> navigation ページの主要なナビゲーションを区切るためのタグ。実行画面上に影響はない。
id="" ID属性 囲っている範囲に名前をつけるための属性。主にリンク先の指定に使用する。
リンク元から指定する際には、必ず付けたidの前に＃を入れる。
<header> ヘッダー 段落やページの導入的なコンテンツを区切るためのタグ。実行画面上に影響はない。
<footer> フッター 段落やページの下部に表示する内容。主に著作権情報やSNSリンクの表示などに用いる。
<small> スモール 文字を注釈的に小さく表示したい際に用いるタグ。
<DOCCTYPE> ドックタイプ 文章宣言と呼ばれる、1行目に入力される情報。HTMLで書かれていると宣言するためのもの。
<HTML> HTML HTML文書を作成するためのタグ。このタグで全体を囲う、一番大枠のタグ。
lang="" ラング属性 <HTML>に付与する、このHTMLで書かれている言語を指定する属性。
<head> ヘッド この文書を解析して活用するプログラムに指定する情報の領域。
<body> ボディ ブラウザに表示する内容を指定する領域。
<meta> メタ情報 特に文章に関する情報を表現するための領域。<head>に用い、文書を管理するための情報を指定する。
charset="" キャラセット <HTML>の文字コードを指定する属性。現在のHTML（Windows 10のバージョン1903（2019年春以降）から）ではUFT-8を使用。
<title> タイトル ブラウザのタブ名やブックマーク時の名称を表示するタグ
name="" ネーム属性 要素に名前を設定するための属性。<meta>属性として付与することで、そのmetaタグがどの要素を持つか決定する。
"description" ディスクリプション 説明を意味する入力部品。<meta>タグにcontent属性で指定することで、
入力した要素が検索結果のサイトタイトル下部に表示される説明として機能する。
content="" コンテンツ属性 <meta>データの内容を指定する属性。
<link> リンク 現在の文書と外部のファイルやページとの関係を結びつけるタグ。
favicon(ブラウザのタブ名横に表示される小さいアイコン)などの指定にも用いる。
rel="" レル(relation) 関係の意。<link>に外部情報の関係性を指定する属性。
後方にhref属性を用いてリンク先を指定する。
<aside> アサイド 本文とは関係の薄い<footer>などを区切る際に用いるタグ。実行画面上に影響はない。
<main> メイン 文書中で主となるコンテンツを区切る際に用いるタグ。
<main>はページ内に1つしか配置できない。実行画面上に影響はない。
<article> アーティクル 記事の意。文書内で自己完結した、また個別に配信や再利用を行うことを意図した情報をまとめるタグ。実行画面上に影響はない。
<time> タイム 日時を表すタグ。タグ内の日時表記は自由だが、
別途後述のdatetime=""属性を用いてyyyy-mm-dd hh:mm の形式で指定する必要がある。
datetime="" デイトタイム yyyy-mm-dd hh:mm の形式を用いて<time>の日時をコンピュータ内に認識させるための属性。
<time>タグ内の時間表記がyyyy-mm-dd hh:mm の場合は不要。
<br> ブレイク(break) 改行をする際に挿入するタグ。終了タグが不要で<br>単体で成り立つ。
段落を作る<p>との混同に注意が必要。
&~; 文字参照 特殊記号やコードで用いる記号などを文字として表示するためのコード。
&と;の間に”キーワード”で指定するものを"文字実体参照"、
”#番号”で指定するものを”数値文字参照"という。
<（小なり）はlt(Less Than)、>（大なり）はgl(Greater Than)で指定する。
<code> コード 文中でプログラムのコードを表現する際に、表現したいコードをこのタグで囲い表示を変化させる。
<p>（文字として挿入）→<p>（コードとして挿入）
<span> スパン 指定したテキストにCSSやJavaScriptを適用し、デザイン変更を行うためのタグ。
主に文中の一部範囲のスタイリングを整える際に使用する。
<div> ディブ(division) 分割の意。囲った範囲を分割しグループ化する働きを持つタグ。
主にまとまった範囲のスタイリングを一括で整える際に使用する。
style="" スタイル属性 主にCSSで用いる属性。付与したタグのスタイルを指定する。
color=""やmargin-left=""など様々な指定を行う。詳細は別述のCSSにて。
<form> フォーム フォームの範囲を指定するタグ。
<input> インプット フォーム内に１行のみの入力欄を表示させるタグ。
後述のtype=""属性で入力タイプを指定する。
type="" タイプ属性 タグのタイプを指定する属性。
<input>に"text"を付与することで、テキストを入力できるようになる。
<textarea> テキストエリア 改行を含む、複数行のテキストボックスを作成する際に用いるタグ。
予め文字を入力する際、字下げがあると空白としてデータに反映されてしまうため注意。
<label> ラベル 要素に対してラベルをつけるためのタグ。
for="" フォア属性 <label>が説明するフォームを指定するための属性。id=""と合わせて用いる。
フォームをラベルで囲うことで、for=""を省略することも可能。
placeholder="" プレイスホルダー テキストボックス内に予め文章を表示させるための属性。
ボックスに入力し始めると消える、文字が薄く表示される等デメリットあり。
"checkbox" チェックボックス チェックボックスを挿入するための入力部品。<input>タグにtext属性で指定する。
<fieldset> フィールドセット フォームの入力項目をグループ化するためのタグ。<fieldset>でフォームを囲うことで、囲われたホームの周りに枠が表示される。
<legend> レジェンド LEGEND：凡例、説明文、銘などの意。グループ化されたフォームの入力項目に対しタイトルをつけるためのタグ。
"radio" ラジオ ラジオボタンを挿入するための入力部品。<input>タグにtext属性で指定する。
name属性で統一の値を指定することで、ラジオボタンとして機能する。
<select> セレクト ドロップリストを表示するためのタグ。<div>タグで囲い要素を分割することで、タグ前にラベルを指定することができる。
後述の<option>タグ、ラベルをまとめて<label>タグで囲うことで、それぞれの要素を関連づける。
<option> オプション ドロップリストの選択肢を指定するタグ。
size="" サイズ属性 指定した要素の表示サイズを指定する属性。<select>タグに対して付与することで、
指定した数値分行が表示される。
また、そのような表示の場合一般にはドロップボックスではなくリストボックスという。
multiple="" マルチプル属性 <select>タグに対し、項目を複数選択できるように指定する属性。
機能上チェックボックスに代用されることが多く、リストボックスへの活用は少ない。
また、属性コードの為基本的にはmultiple="multiple"となるが、属性=名前の場合は=""を省きmultipleのみ付与で指定可能。
<button> ボタン ボタンを表示するタグ。<div>タグで閉じて用いる。
<input type="submit" value="表示名">を用いることでも表示できるが、<button>がより汎用性が高い。
disabled ディスエイブルド属性 付与したタグの機能を止める属性。<button>に付与することでボタンがグレーアウトする。
必須情報を入力することで有効化になる、などの動作はJavaScriptで制御する。
<value=""> バリュー属性 初期値やラベル、パラメータなどを指定するための属性。<input type="text">に付与することで、ページを開くと予めセットした値が表示されるようになる。
<checked=""> チェックド属性 <checkbox>に付与することで、予めチェックがつくようになる属性。
属性=名前のため、=""は省略する。
<input type="radio">の複数項目に付与してしまった場合、最後に付与した項目のみ有効となる。
<selected=""> セレクテッド属性 <option>タグに使用する属性。付与すると予めリストが選択される状態になる。
属性=名前のため、=""は省略する。
"password" パスワード パスワードを入力するフォームで使用する入力部品。
<input type="password">として指定することで、入力した値が伏せ字になる。
"number" ナンバー 数値入力に特化した入力部品。<iput type="number">と指定することで、
数値の上下を矢印で指定することが可能。
見え方や操作がブラウザ依存、古いブラウザでは動かない場合がある等注意が必要。
"date" デイト 日付入力に特化した入力部品。<iput type="date">と指定することで、
入力項目にカレンダーを表示しより直感的に日付を入力できる。
見え方や操作がブラウザ依存、古いブラウザでは動かない場合がある等注意が必要。
